# SuperWarehouse Local

Локальное JavaFX-приложение складского учета с регистрацией, авторизацией и SQLite-базой.

## Запуск

Требуется JDK 21 и Maven.

```bash
mvn clean javafx:run
```

## Первый вход администратора

Логин: `admin`

Пароль: `Admin123!`

## Где хранится БД

SQLite-файл создается автоматически в:

`%USERPROFILE%\.superwarehouse\warehouse.db`

## Функции

- регистрация и вход;
- роли ADMIN и USER;
- добавление, редактирование и удаление товаров;
- поступление, продажа и списание;
- история операций;
- поиск, фильтрация и сортировка;
- контроль товаров, срок годности которых истекает в ближайшие 7 дней.

Если раньше уже запускалась старая версия, удалите файл базы `%USERPROFILE%\.superwarehouse\warehouse.db`, чтобы схема создалась заново.
