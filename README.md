# SuperWarehouse

Локальное JavaFX-приложение для складского учёта товаров, управлением остатками и историей операций. SuperWarehouse помогает добавлять товары, отслеживать сроки годности, фиксировать поступления, продажи и списания, а также управлять пользователями с разными ролями доступа.

## Технологии

* Java 21
* JavaFX 21.0.2
* SQLite (sqlite-jdbc 3.53.0.0)
* Maven
* JUnit Jupiter 5.10.2

## Возможности

* добавление, редактирование и удаление товаров;
* поступление, продажа и списание с автоматическим пересчётом остатков;
* история всех операций с фильтрацией по типу;
* поиск, фильтрация по категории и единице измерения, сортировка;
* контроль товаров, срок годности которых истекает в ближайшие 7 дней.

## Установка

1. Склонируйте репозиторий:

```bash
[git clone https://github.com/your-username/SuperWarehousLoc.git
cd SuperWarehousLoc]
```

2. Убедитесь, что установлен JDK 21 и Maven.

## Быстрый старт

Запуск из терминала:

```bash
mvn clean javafx:run
```

Для Windows PowerShell:

```powershell
mvn clean javafx:run
```

## Тесты

В проекте есть юнит-тесты для логики валидации товаров и утилиты хэширования паролей.

Быстро запустить тесты можно прямо в IntelliJ IDEA:

1. Откройте папку `src/test`.
2. Выберите нужный тестовый класс или весь пакет.
3. Нажмите зелёную кнопку запуска рядом с тестом.

Запуск тестов из терминала:

```bash
mvn test
```

Для Windows PowerShell:

```powershell
mvn test
```
